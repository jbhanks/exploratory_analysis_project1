To use these scripts:
1. Download the file "https://d396qusza40orc.cloudfront.net/exdata%2Fdata%2Fhousehold_power_consumption.zip" to a convenient location
2. Set the working directory in R to the directory containing the downloaded file.
3. Run the scripts (plot1.R, plot2.R, plot3.R, plot4.R)
4. The graphs will be saved into the working directory you set earlier.
